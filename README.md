# Lindys SCPI Server (lindyscpiserver)
Local server purposed to simulate measurement tool like osciloscope or Multimeter. It simulates measurement data and can be started from CLI. 

## SCPI Commands
At this moment it simulates Keysight SCPI general commands for DMM. 

## Simulated device
At this moment it simulates only Voltage measurement of 12 Volt AGM Lead battery.
